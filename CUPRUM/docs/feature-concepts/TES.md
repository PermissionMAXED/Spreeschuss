# Family TES — Tesla & Electric Combat (`tesla_combat`)

Global sequence 79–94 · 16 features (12 core / 4 stretch) · core wave W7, stretch W15 · catalog family string: `tesla_combat`

Offense, area denial and counterplay built on U09 Tesla Turrets and the **canonical U03 Shock effect** (all stun references below are to U03's registration; the W12 FX family only extends it). Every arc weapon draws from PWR storage; every offensive option has a published counter (TES-10 weave, TES-12 EMP, TOOL-17 insulation).

**Balance constants:** turret baseline 60 Cg/shot, 5 damage, 16-block range, 20-tick cooldown; upgrades trade burst/coverage, never Cg-per-damage efficiency. EMP (TES-12) disables hostile-owned Cuprum electrics for 200 ticks in a 12-block radius with a 600-tick per-thrower cooldown. Fence discharge 30 Cg, 4 damage. TES-01 cards are per-turret entity-class filters; team-wide friend/foe policy belongs to SHD-13 banners — deliberately separate scopes.
**Performance budget:** turret target acquisition uses dome-style cached entity sets, ≤0.05 ms/tick per turret at 32 candidates; all active arcs batch into one render submit pass (T1) or one particle batch (T2/T3).
**Security contract:** turret filter writes, mortar aiming and baton use are C2S-validated (ownership, ≤8-block GUI distance, 4/s rate limit); EMP throws are logged with thrower UUID and default to hostile infrastructure only (see TES-12); turrets never target players unless a players-mode card is installed by the turret owner.
**Persistence contract:** turret filters, mine arm states and stored charge persist in NBT across unload/restart; a disabled (EMP) state expires by game-time comparison, so restarts do not extend it.
**Cross-family interfaces:** TES-05 Arc Welder is the standard repair tool for GOL golems and machines; TES-11 feeds the U04 capture network; FX-06 Voltaic Edge marks targets for turret priority; TES-14 tones are reused by DEC-15.
