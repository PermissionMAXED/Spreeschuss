# Family GEN — Worldgen, Biome Regions & Structures (`worldgen`)

Global sequence 205–218 · 14 features (10 core / 4 stretch) · core wave W11, stretch W15 · catalog family string: `worldgen`

Places that teach, reward and seed the copper ecosystem. All structures are static jigsaw datapack assets **within Minecraft's 128-block jigsaw placement cap — no hidden custom infinite worldgen**; no runtime feature mutates worldgen after placement except the two explicitly bounded events (GEN-06 growth, GEN-14 crater), both transactional block placements in loaded chunks.

**Balance constants:** structure loot gives starter parts and blueprints, never finished endgame machines; GEN-10 canyon ore density (≥3× mean) is offset by MOB-03 mite pressure and MOB-02 oxide creepers in GEN-13 regions.
**Performance budget:** worldgen assets are static datapack entries with zero runtime tick cost; GEN-06 buds advance on random tick only; GEN-14 places one pre-computed template in a single tick.
**Security contract:** GEN-14 targets only newly generating, unclaimed natural chunks inside the world border, replaces only natural-whitelist blocks and aborts (0 changes) if the template intersects any player-placed block — it can never overwrite player builds.
**Cross-family interfaces:** GEN-01 foundries spawn MOB-05 junk golems; GEN-05 malachite shards cut into lenses consumed by DEC-13 holo-signs (and an optional alternative gauge-lens recipe noted in QOL-12); GEN-08 camps house MOB-08 electricians and stock OXI-02 alloy; GEN-09 caches are marked on TOOL-04 survey maps; ADV-08 map compasses point to GEN-01.
