# Family SHD — Storm Shield Extensions (`shield_tech`)

Global sequence 65–78 · 14 features (11 core / 3 stretch) · core wave W7, stretch W15 · catalog family string: `shield_tech`

Upgrades, modulators and counterplay around the U01–U03 Storm Shield vertical slice. Nothing here duplicates U01/U02/U03; every entry modifies, specializes or reads the dome. Shock references are to the canonical U03 registration.

**Balance constants:** dome upkeep is ceil(0.5·R²) Cg/t (R8=32, R12=72, R16=128, R24=288); projectile intercept costs 200 Cg + 40 Cg per block/tick of projectile speed; mob repulse costs 150 Cg with a 10-tick per-entity cooldown. Modulators trade coverage for upkeep with integer rounding: kinetic-only upkeep = ceil(base × 0.6) Cg/t (20/44/77/173 for bases 32/72/128/288), biotic-only = ceil(base × 0.7) Cg/t (23/51/90/202), weather ×1.0 but capture-enabled. **Economy note (per PWR formulas):** upkeep is storage-driven — an R8 dome (32 Cg/t) consumes one captured strike (270,000 Cg) in ≈8,400 ticks (7 min) of continuous uptime; shields are siege equipment backed by capture and generator banks, not free always-on cover.
**Performance budget:** all SHD logic rides the existing U01 dome tick (no second entity scan); modulators are controller flags, not extra block entities; ripple/echo payloads reuse the U02 ring buffer (max 16 concurrent ripples).
**Security/ownership contract:** dome configuration (modulators, hue, window triggers) requires owner or team membership, permission node `cuprum.shield.configure` when a permissions provider is present, else owner/OP; all C2S config payloads are distance-checked ≤8 blocks and rate-limited 4/s.
**Persistence contract:** controller stores charge, mode flags, hue and team in NBT; the field exists only while the controller chunk ticks; state is identical after unload/reload/restart.
**Cross-family interfaces:** SHD-07 consumes PWR-18 priority tiers; SHD-13 consumes scoreboard teams (a dome-wide friend/foe policy, distinct from TES-01 per-turret entity-class filters); SHD-14 uses the same Glowing mechanic as TOOL-13; SHD-06 mode patterns are shape-coded so shield status stays readable without color.
