# Family TOOL — Tools & Gadgets (`gadgets`)

Global sequence 157–178 · 22 features (17 core / 5 stretch) · core wave W10, stretch W15 · catalog family string: `gadgets`

Handheld utility extending U13 fans, U14 grapple, U15 detector, U16 plates and U18 XP drones. Chargeable gadgets share one battery/charging interface (PWR-19/PWR-20) and one HUD charge-bar widget.

**Balance constants:** gadget charge costs are sized so a 50,000 Cg pack sustains a session of casual use; movement gadgets stay below elytra ceiling (horizontal mobility, never flight); info tools reveal but never auto-act.
**Performance budget:** gadget hooks are event-driven (0 per-tick inventory scans); the magnet glove considers ≤32 item entities per pull tick; ghost projections rebuild client meshes only on blueprint change; survey printing is amortized per the scan-safety contract.
**Scan-safety contract (binding):** all scanning tools run bounded, time-sliced scans on the main server thread — TOOL-04 scans at most 4,096 positions total at ≤256 positions/tick and never touches the Level off-thread; TOOL-10 captures at most 32³ blocks with the same slicing rule.
**Permission contract:** TOOL-10 scanning and TOOL-03 remote lever actuation require block-edit permission at the target position (claim/protection hooks); TOOL-03 pulls only item entities, never containers' contents; remote actuations are rate-limited to 4/s per player.
**Persistence contract:** blueprint items, survey maps and calibration states are item components that survive save/reload; ghost projections rebuild client-side from the projector block entity on chunk load.
**Cross-family interfaces:** TOOL-04 Survey Maps are readable by GOL-06 miner routines and mark GEN-09 caches; ghost previews (TOOL-11) reuse the U22 handbook multiblock preview renderer; TOOL-14 writes U16 plate block entity settings; TOOL-12 converts U18 drone deposits; TOOL-08 and QOL-11 recipes consume WEA-06 charge shards.
