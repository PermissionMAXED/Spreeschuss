# Family EXO — Exoskeleton & Armor Modules (`exo_modules`)

Global sequence 179–192 · 14 features (11 core / 3 stretch) · core wave W9, stretch W15 · catalog family string: `exo_modules`

Modular expansion of U08: chassis tiers, socketed limb modules and the heat economy that keeps stacked modules honest. Back-slot handling is the mod's own attachment logic (Trinkets/Accessories have no 1.21.9 builds — verified absent).

**Balance constants:** chassis power budgets cap total module draw at 100/200/300 Cg/t for tiers I/II/III (storage-fed burst loads per the PWR economy formulas). **Heat model:** heat rises 0.3/tick at full draw; throttling starts at exactly 120 heat; the EXO-11 coolant loop dissipates 0.2/tick (net 0.1/tick), so full draw throttles at 1,200 ticks with the loop versus 400 ticks without. TES-12 EMP disables all modules for 200 ticks; the glider is strictly below elytra (6:1 glide, no rocket boost).
**Performance budget:** module effects are event-driven attribute/ability hooks, no per-tick inventory scans; wearer sync piggybacks the player attachment sync at ≤1 payload/s steady-state.
**Persistence contract:** installed modules, stored charge and heat persist in item components across death (kept in the chassis item), save/reload and restart; EMP-disabled state expires by game-time comparison.
**Cross-family interfaces:** charges via PWR-19/PWR-20; EXO-07 gains altitude from U13 fan updrafts; EXO-09 embeds the U15 detector; EXO-10 consumes OXI-08 bath-applied patina; U17's backpack shield occupies the same back socket as EXO-06/EXO-07 — an exclusive loadout choice.
