# Family DEC — Functional Decor & Building (`decor_building`)

Global sequence 245–260 · 16 features (12 core / 4 stretch) · core wave W13, stretch W15 · catalog family string: `decor_building`

Building-block sets and furniture with mechanics — per the INDEX "spectacle stretch" policy, every core entry carries a concrete building, network or status function (connected building sets, charge conduction, light/heat output, information display); pure spectacle pieces are stretch.

**Balance constants:** decorative conductors (DEC-02, DEC-10) carry reduced throughput so they complement, never replace, PWR wiring; light/heat fixtures consume charge so aesthetics load the grid (lamp bands and radiator draw are sized against the 5 Cg/t passive baseline).
**Performance budget:** connected-texture resolution is client-side model logic with zero server cost; animated fixtures cap at 4 animated block entities per chunk section before switching to static LODs.
**Persistence contract:** sign/board text, canvas hues, drawer contents and carillon scores persist in block-entity NBT across unload/restart; board page edits are C2S-validated at ≤8 blocks.
**Cross-family interfaces:** DEC-05/DEC-06 draw from PWR networks; DEC-07 carillons read GOL-03 punch cards as note scores; DEC-08 boards can optionally display PWR-23 teleprinter feeds when that stretch feature is present (never required); DEC-13 holo-signs consume GEN-05 malachite lenses; DEC-11 canvas consumes OXI-04 patina extract.
