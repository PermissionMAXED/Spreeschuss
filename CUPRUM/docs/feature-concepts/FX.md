# Family FX — Status Effects, Enchantments & Consumables (`effects_enchants`)

Global sequence 219–234 · 16 features (13 core / 3 stretch) · core wave W12, stretch W15 · catalog family string: `effects_enchants`

Advanced effect tuning, enchantments and consumables layered on the **canonical registrations owned by U03 and U06**: U03 registers Shock (interrupt + Slowness IV, 40 ticks; repeat applications within 200 ticks last 20 then 10 ticks; 10-tick per-source re-application cooldown) and U06 registers Corroded (−10% armor effectiveness per level, cap II = −20%, 200 ticks). Earlier OXI/TES/MOB rows reference those registrations directly; FX-01/FX-02 below are genuinely additional W12 extensions, not re-registrations.

**Balance constants:** Overload Shock (FX-01) is a separate, stackable heavy-source effect (60 ticks, blocking disabled) that applies on top of the base U03 Shock and shares U03's diminishing-return tracker — it never replaces or re-registers U03; Deep Corrosion (FX-02) adds armor-durability bleed at Corroded II without raising the level cap. Enchantments are data-driven 1.21.9 enchantment JSON.
**Performance budget:** 500 concurrently active Cuprum effect instances add at most 0.20 ms/server tick averaged over 1,000 ticks, enforced by the W14 perf gate `w14_fx_effect_budget`; effect ticking uses vanilla MobEffect infrastructure; chain lookups (FX-05) query the cached dome/turret entity sets, never fresh AABB scans.
**Persistence contract:** effect durations, curse states and stored-charge components persist via vanilla effect saving and item components.
**Cross-family interfaces:** U03 Shock and U06 Corroded are applied by TES weapons, MOB-02 and OXI-09/10; FX-06 marks feed U09 turret priority; FX-12 curses come from MOB-09 rogue units and are cleansed at the OXI-13 station; FX-14 brews consume OXI-16 conductive paste (MOB-10 gel is an optional double-duration substitute, never required).
