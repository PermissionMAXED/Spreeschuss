# Family ADV — Advancements & Progression (`progression`)

Global sequence 235–244 · 10 features (9 core / 1 stretch) · core wave W12, stretch W15 · catalog family string: `progression`

The connective tissue: advancement trees, quest chains, unlock gates and museum meta-goals that pace all other families. Design rule: gates only ever unlock additions (modes, discounts, recipes for new items); no gate ever retroactively locks a previously craftable base item.

**Balance constants:** ADV-04 requires the MOB-04 Storm Heart to craft the U21 Overdrive Core — the base weather manipulator is craftable without any boss kill; ADV-07 mastery discounts cap at 10% so veterans save charge but newcomers are never priced out.
**Performance budget:** advancement triggers are event-driven criteria (0 polling loops); mastery counters write ≤1 attachment update per machine operation; quest-step checks run at most 1 evaluation per player per 20 ticks; the full tree JSON loads once at datapack reload (≤50 ms parse budget).
**Persistence contract:** quest state, mastery counters and bank contents are player/world attachments that survive relog and restart; ADV-10 XP banking is capped and never duplicates XP.
**Cross-family interfaces:** ADV-01 grants pace the whole catalog; ADV-05 consumes GEN structure artifacts and MOB-12 curator bounties reference the same artifact set; ADV-08 compasses point to GEN-01 foundries; ADV-09 elder trades activate after the ADV-04 overdrive milestone and sell blueprints for PWR-14, SHD-10 and TES-08.
