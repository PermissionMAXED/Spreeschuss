# Family OXI — Oxidation Metallurgy & Materials (`oxidation_metallurgy`)

Global sequence 47–64 · 18 features (16 core / 2 stretch) · core wave W6, stretch W15 · catalog family string: `oxidation_metallurgy`

Materials, processing machines and controlled-patina crafting that deepen U06/U07/U20. **U06 owns and registers the canonical Corroded effect** (−10% armor effectiveness per level, cap II = −20%, 200 ticks); every OXI weapon references U06's registration, never later FX rows.

**Bootstrap rules:** the Alloy Kiln (OXI-03) crafts from vanilla materials only (bricks + copper ingots + furnace), and Patina Extract (OXI-04) has a vanilla-material W6 bootstrap recipe (4 stage-3 oxidized cut copper + 1 glass bottle), so the family has no circular entry requirement. The kiln produces Cuprite Alloy (OXI-02); the alloy is the kiln's registered output, never a prerequisite for building the kiln. Casting components (OXI-15) are consumed only by recipes in W7+ families; PWR (W5) machines use plain copper/iron recipes.
**Balance constants:** kiln alloy op 200 ticks unpowered at 0 Cg; optional fast path debits exactly 500 Cg upfront and finishes in 100 ticks (no per-tick draw). Refinery 40 Cg/t, 160 ticks/op; mill 30 Cg/t, 100 ticks/op; weathering chamber 20 Cg/t, 1 stage per 1,200 ticks; patina transfer 50 Cg per stage moved. Ore doubling is copper-only. Machines are storage-fed burst loads per the PWR economy formulas.
**Performance budget:** item oxidation advances on random-tick and event hooks only, never per-tick inventory scans; the Weathering Chamber changes at most 1 stage per 100 ticks per slot.
**Persistence contract:** machine progress, reservoir contents and wax states are block-entity NBT; all survive unload/reload/restart.
**Cross-family interfaces:** OXI-15 components are standard intermediates for W7+ recipes; OXI-04 patina extract is the aging agent consumed by DEC-11 (MOB-06 later adds an optional feather-based alternative recipe); OXI-16 conductive paste is the FX-14 brewing ingredient; OXI-13 is the wax/de-wax workstation (later waves may add optional cleansing functions, see FX-12).
