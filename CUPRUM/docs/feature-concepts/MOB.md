# Family MOB — Mobs & Entities (`creatures`)

Global sequence 193–204 · 12 features (9 core / 3 stretch) · core wave W11, stretch W15 · catalog family string: `creatures`

Creatures that give the copper ecosystem pressure, reward and personality. All animation is code-driven keyframes (GeckoLib has no 1.21.9 build — verified); nothing here duplicates the vanilla copper golem, which is only extended via U12/GOL. Effect references are to the canonical U03 Shock and U06 Corroded registrations.

**Balance constants:** rust mites threaten only unwaxed machines, so OXI-05 wax is the cheap defense; the Storm Elemental (MOB-04) is the single boss and spawns only in genuine thunderstorms at altitude; its Storm Heart unlocks the U21 overdrive mode defined in ADV-04 (the base manipulator is never gated).
**Performance budget:** ambient mobs cap at 2 (wisp) or 4 (peacock) per chunk; boss fight effects honor QOL-04 reduced-effects caps; no mob paths wider than vanilla illager patrol logic.
**Persistence contract:** jarred wisps, hacked rogue units, golem conversions and boss fight state persist across unload/reload/restart.
**Cross-family interfaces:** MOB-05 junk golems populate GEN-01 foundries; MOB-08 trades sell GOL-03 blank cards and TOOL-04 survey maps; MOB-09 hacking uses GOL-12 and sources FX-12 curses; MOB-06 adds an optional feather-based alternative recipe for OXI-04 extract.
